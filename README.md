# IDS-software
This repository contains the base software implementation for the Intrusion Detection System for LoRaWAN Networks, developed during the final project of MSc in Informatic and Computers Engineering, by the student Carlos Tavares (number 48725) of ISEL Lisbon School Engineering, in 2024/25 school year


# init.py
Sets up database schema through SQL tables


# GUI.py
Implements a Graphical User Interface (GUI) for an Intrusion Detection System using PySimpleGUI. It interacts with a CrateDB database to retrieve, display, and manage data related to LoRaWAN sensors and network events. The system helps the user to view network metrics, handle potential doubts, and flag possible network intrusions.
